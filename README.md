# ProdigyInfoTech_DS_Task_4
📊 Sentiment Analysis of Social Media Data
Task 4 – Prodigy Infotech Data Science Internship
📌 Project Overview
This project focuses on analyzing and visualizing sentiment patterns in social media data to understand public opinion and attitudes toward specific topics or brands.

The goal is to classify text data into Positive, Negative, and Neutral sentiments and identify trends using data visualization techniques.

🎯 Objectives
Perform data cleaning and preprocessing on social media text data
Apply Natural Language Processing (NLP) techniques
Analyze sentiment patterns
Visualize public opinion trends
Understand how users react to specific topics or brands

📊 Results
Sentiment analysis was performed on the twitter.csv dataset to classify tweets into Positive, Negative, and Neutral categories. The majority of tweets were Positive, indicating a generally favorable public opinion. Neutral tweets represented informational or unbiased responses, while Negative tweets were comparatively fewer. Visualizations such as bar charts, pie charts, and word clouds clearly illustrated sentiment distribution.

🚀 Conclusion
The sentiment analysis of Twitter data effectively revealed overall public opinion and attitude toward the selected topic/brand. Most tweets reflected a positive sentiment, indicating a generally favorable public perception.
